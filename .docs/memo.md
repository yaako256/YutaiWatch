# メモ
- discordへの通知は、一度に2項目取得した時、それぞれを送信してもいいが、勘違いなどを防ぐため、1つだけ送信することにする。
- discordは単純に本文だけを送る方法もあるが、`embed`という、「カード形式」のメッセージも送れる。こっちの方がカラフルになったりして面白そう

- サイトのURLはどこで定義しよう

- アプリの使用となる部分は定数として.rsに、設定項目は.tomlに設定を分ける。

- HTMLパースで使うクラス名の定義とかはどこで行おう。

- configのloaderが雑魚いせいで、.envが横長くなっている。いつか`,`以外でパースして取得させたい

- CLIで、init、monitorのほかにresizeみたいなやつを作って、1か月に1回くらいdetect_historyとupdate_historyの古い部分を消すのがいい気がする。その時はconstantsのfileに定数を追加する。

- discordクレートをnotifyクレートにして、sharedにトレイトを書いて、ほかの媒体もできるようにしたい。
